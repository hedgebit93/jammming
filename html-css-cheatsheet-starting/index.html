CSS font-weight Property

The CSS font-weight property declares how thick or thin the text is. Numerical values can be used to set the thickness of the text ranging from 100 to 900 and only accepts multiples of 100.

The default value is normal while the default numerical value is 400.

Any value less than 400 whill have text appear lighter than the default. This sentence has been set to a numerical value of 200.

Any value greater than 400 will appear bolder. This sentence has been set to a numerical value of 800.
